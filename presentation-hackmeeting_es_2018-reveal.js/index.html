LibreMesh: modulos

  • lime-system -- conexion layer 1: AP + IEEE802.11s, roaming y autoconfiguracion
  • lime-proto-batadv -- BATMAN adv, routeo de layer 2
  • lime-proto-bmx6 -- BMX6, routeo de layer 3
  • dnsmasq-lease-share -- compartir la tabla de leases para evitar collisiones DHCP
  • bmx6-auto-gw-mode -- compartir acceso internet
  • lime-app -- nueva interfaz web

Muchos mas modulos para suportar cada tipo de red

  • lime-hwd-usbradio -- soporte para radios USB
  • lime-proto-babeld -- BABELd, routeo layer 3
  • lime-proto-olsr2 -- OLSR2, routeo layer 3
  • lime-proto-bgp -- BGP, conexion inter redes
  • pitbull -- splash page o captive portal
  • tinc -- soporte para VPN
